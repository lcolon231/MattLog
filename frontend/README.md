# MatLog Frontend

React + Vite frontend for MatLog.

## Setup

```powershell
npm install
npm run dev
```

The app runs at `http://localhost:5173`.

## API URL

By default the frontend calls `http://localhost:8000`.

To point it somewhere else, create a `.env` file:

```text
VITE_API_URL=http://localhost:8000
```

## Available Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`
